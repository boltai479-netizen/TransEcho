## 🪟 What TransEcho does

TransEcho captures system audio on your computer, translates speech in real time, and shows subtitles on screen. It can also speak the translated text back through voice output.

It is built for daily use on Windows and macOS. You can use it for meetings, videos, classes, calls, and live talks.

## 🔊 How to use it

### 1. Capture system audio

Select system audio capture inside the app. This lets TransEcho listen to sound that plays on your computer, such as:

- Meeting audio
- Video sound
- Online classes
- Live streams
- Calls in speaker mode

### 2. Pick your languages

Choose the language you hear and the language you want to read or hear back. TransEcho sends the audio through speech to text, then into translation, then into subtitles and voice output.

### 3. Turn on subtitles

Subtitles appear on screen in real time. This helps when you want to read the translation while listening to the source audio.

### 4. Turn on voice output

If you want spoken translation, enable text to speech. TransEcho will speak the translated text after it is processed.

### 5. Adjust the view

You can keep the subtitle window visible while you work. This is useful for long sessions and for following fast speech.

## 🔧 Common problems

### The app does not start

- Run the downloaded file again
- Check that Windows did not block the file
- Try the newest release
- Restart your PC and open the app again

### No audio is captured

- Make sure system audio capture is enabled
- Check the selected playback device
- Increase the computer volume
- Close other apps that may take the audio source

### Subtitles do not appear

- Confirm that the app is listening to the right source
- Check your language selection
- Restart the capture session
- Make sure the subtitle panel is open

### Voice output is silent

- Check your speaker or headset
- Make sure voice output is turned on
- Raise the output volume
- Test with another audio device
